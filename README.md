# AI Blind Glasses 👓

Real-time smart glasses assistant for visually impaired users using **OpenAI Vision API** + **Python**.

The system uses a camera to capture live images, sends them to OpenAI, and speaks scene descriptions such as:

- Person in front of you  
- Chair on left side  
- Door ahead  
- Stairs nearby  
- Car approaching  

---

# Features

✅ Real-time scene understanding  
✅ Voice feedback  
✅ Detects obstacles & surroundings  
✅ Works with webcam / USB camera / Raspberry Pi camera  
✅ Easy to customize
